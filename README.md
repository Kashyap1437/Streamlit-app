# AI Business Analyst Agent

## Project Overview
AI Business Analyst Agent is an interactive Streamlit app that allows users to upload a CSV file, analyze business performance, view KPIs, create charts, run SQL queries, and generate business recommendations.

The project is designed to support business analyst, data analyst, operations analyst, and product analyst workflows.

## Tools Used
- Python
- Streamlit
- Pandas
- SQLite
- SQL

## Features
- Upload CSV files
- Preview dataset rows and columns
- Check missing values
- View summary statistics
- Generate KPI cards
- Create business charts
- Ask basic business questions
- Run SQL queries on uploaded data
- Generate AI-style business recommendations

## Example Questions
- What is the total Sales?
- What is the average Profit?
- What are the top categories by Quantity?
- What should the business focus on?
